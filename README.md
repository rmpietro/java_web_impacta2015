# java_web_impacta2015
Código fonte dos exercícios de laboratório realizados e exercitados em sala de aula

Senhores, em cada pasta relativa ao Capítulo específico existirão uma ou mais pastas relativas aos laboratórios das aulas.
Essas pastas são, na verdade, projetos prontos do Eclipse para serem importados.
Basta realizar a importação do projeto e começar a trabalhar! 

Lembre-se de que existe a necessidade de ter configurado e adicionado em seu workspace o Apache Tomcat 7 e o JDK 7 instalado na máquina. Caso você tenha instalada outra versão de Java na máquina ou do servidor Tomcat, será necesessário ajustar as configurações do projeto para essas versões. Uma vez ajustadas, os laboratórios executam sem problemas.

É indiferente a versão de Eclipse a ser usada, desde que compatível com o JDK 7.

Grande abraço a todos e bons estudos!
